# HTML & CSS Landing Pages Collection

## Project Overview

This repository contains multiple landing pages created using HTML and CSS.
The purpose of these projects was to practice layout design, styling, and responsive web development.

The repository includes three different landing page designs.

## Pages Included

### 1. Food Landing Page

Files:

* food.htm
* food.css

Description:
A simple landing page design for a food-related website. This page focuses on layout structure and basic CSS styling.

### 2. Product Reviews Page

Files:

* reviewsOfProducts.htm
* reviewsOfProducts.css

Description:
A landing page that displays product reviews and information. This project demonstrates structured content layout and styling.

### 3. Responsive Website (Main Project)

Files:

* index.html
* style.css

Description:
A fully responsive website designed using HTML and CSS. The layout adapts to different screen sizes including desktop and mobile devices.

Features include:

* Responsive layout
* Structured sections
* Modern UI styling
* Mobile-friendly design

## Technologies Used

* HTML5
* CSS3
* Responsive Design (Media Queries)
